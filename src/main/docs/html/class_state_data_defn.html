File Description: Top-level XML parsing file for the state data. This definition will construct objects to help set the state targets and control modes.

This parsing leverages the 3rd party Open Source Pugixml library (https://pugixml.org/).

The state definition XML files are in: /home/lvuser/config/states/XXX.xml where the XXX is the mechanism name.
